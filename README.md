# Yi패ka패

A simple, responsive, and accessible product-listing application built with React.

**Feature Highlights:**

-   Category filtering
-   Price sorting (High to Low / Low to High)
-   Favourites with `localStorage` persistence
-   Light/dark theme
-   Responsive layout
-   Accessible markup
